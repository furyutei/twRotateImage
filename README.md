■ twRotateImageとは？
---
[Twitterの公式Webサイト](https://twitter.com/) で、画像を回転表示するユーザースクリプトです。  
マウスカーソルをあわせた状態で[R]キーを押すと、画像が90度ずつ回転します（デフォルトは時計回り・[Shift]＋[R]で反時計回り）。
